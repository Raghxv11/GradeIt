Final Project Report

The Project Report is a professionally formatted report that combines your three sections into a formal written document. You need to complete the following:

  1. Create a cover page with your name and your Innovation Ideation title. (10 pts)
  2. EDIT FROM PREVIOUS FEEDBACK and compile your three sections. (20 pts)
  3. There should be no grammar or spelling mistakes. (20 pts)
  4. Make sure the formatting is consistent. (10 pts)
    • All headers at a similar level use the same size and style Font.
    • Each section Header has the same style
    • Each subheader has the same style
  5. Make sure all Figures are labeled with captions by name and the correct number with callouts and explanations in the text. (20 pts)
  6. List all references and sources used in a References section (20 pts)

Deliverable:

Single PDF document containing the above uploaded to Canvas
